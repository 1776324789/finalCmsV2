cms-slider属性
属性名称 说明 默认值
data 对应后台中list中的序列号，加载层级最高。 null
dev 对应后台中list中的序列号，加载层级最低。 null
page 加载列表内容的页面，配合limit使用 1
limit 加载列表内容条数，配合page使用 null
auto 是否自动轮播图片，值为切换时间间隔（毫秒） null
width 轮播图宽度 null
height 轮播图高度 null
slider:bind data绑定cms-node,详情查看下方slider:bind 属性 null
轮播图在未设置高度和宽度的情况下，高宽默认为0 data加载顺序:cms-node.data   >   cms-component .data  >  url?data=dataId  >   cms-node .dev
slider:bind 属性
绑定轮播图当前页的id至相同参数的相关组件，使用在不同组件有不同效果，详情查看示例。

slider:bind涉及子标签
子标签名称 功能 注意事项
list-name 加载列表的名称 无
list-note 加载列表的备注 无
list-cover 加载列表的封面 自身可以看作为img标签，所有写在该标签的style、class都会继承到创建的img子标签中，然后移除自身标签。
list-node-size 包含内容的数量 无
list-template 跳转到列表页面 该标签包裹的部分点击后跳转
list-list 加载列表的子列表 通过创建cms-list的方式加载子列表，可嵌套。（cms-list参与加载，但不参与到最终的渲染）
list-node 加载列表的子内容列表 通过创建cms-node的方式加载子内容，可嵌套。（cms-node参与加载，但不参与到最终的渲染）
list-pagination 子内容列表中的翻页组件 该标签只能在list-node中使用
