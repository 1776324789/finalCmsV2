欢迎使用 FinalCMS

一个现代化的、灵活的、易于使用的内容管理系统

快速搭建

FinalCMS 提供了丰富的组件和灵活的配置选项，帮助您快速构建网站：

  • 使用 cms-list 和 cms-node 管理内容
  • 支持自定义组件和模板
  • 响应式设计，支持多端适配

核心组件

系统提供了多个核心组件，满足常见的内容管理需求：

  • cms-list：列表展示组件
  • cms-node：内容节点组件
  • cms-component：自定义组件

全局搜索

强大的搜索功能，帮助用户快速找到所需内容：

  • 支持多字段搜索
  • 实时搜索结果
  • 自定义搜索范围

开发工具

为开发者提供便捷的开发工具和文档：

  • 详细的API文档
  • 组件开发指南
  • 示例代码和模板
