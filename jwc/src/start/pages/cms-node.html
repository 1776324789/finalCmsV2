cms-node标签
cms-node 加载节点内容数据
cms-node属性
属性名称 说明 默认值
data 对应后台中list中的序列号，加载层级最高。 null
dev 对应后台中list中的序列号，加载层级最低。 null
pagetitle 将文章标题引用为页面标题 null
data加载顺序:cms-node.data   >   cms-component .data  >  url?data=dataId  >   cms-node .dev
cms-node子标签
子标签名称 功能 注意事项
node-title 加载节点内容的标题 无
node-top 加载节点内容是否为置顶内容，是则该标签会注入"置顶"，否则为空 无
node-date 加载节点内容的创建时间 默认格式YYYY-MM-DD HH:mm:SS
node-note 加载节点内容的备注 无
node-link 加载节点内容的外链,本身不带任何内容，点击后跳转至外链 无
node-clicks 加载节点内容的点击量 无
node-cover 加载节点内容的封面 自身可以看作为img标签，所有写在该标签的style、class都会继承到创建的img子标签中，然后移除自身标签。
node-file 加载节点内容的文件下载链接 默认填充文件名，点击即下载
node-template 内容详情页的跳转链接，也就是父list的内容页链接。 无
node-content 加载节点的内容，注入为html 异步加载，节点的内容和节点的基础数据不在一起加载
node-parent 加载节点的父list 可以视作为cms-list标签
node-prev 点击跳转至上一个文章，会将上一个文章的标题注入此元素，点击即跳转，跳转规则和node-template一致 当不存在则提示：没有了
node-next 点击跳转至下一个文章，会将下一个文章的标题注入此元素，点击即跳转，跳转规则和node-template一致 当不存在则提示：没有了
所有写在页面中的组件标签都可以视作一个div，组件标签自身不带任何样式。
node-date
属性名称 说明 默认值
format 时间格式化:mm为分钟，MM为月,其他单位不区分大小写 YYYY-MM-DD HH:mm:SS
node-template
属性名称 说明 默认值
newpage 是否在新页标签打开页面 false
node-link
属性名称 说明 默认值
newpage 是否在新页标签打开页面 false
node-content
属性名称 说明 默认值
line 显示的行数，会自动去掉格式，转换为纯文字，并自动加上省略号 null
line:medium 在缩略显示时，是否显示媒体，false时会将标签替换为代替文字【图片】【视频】 true
empty 当content中的内容为空或者无效内容（只有空格和回车）时，显示的替代内容 null
使用示例
这是一段很长的内容... 这是第二行内容... 这是第三行内容... 这是第四行内容但不会显示... 第一行文字 【图片】 【视频】 更多内容...
- line 属性会移除所有 HTML 标签，只保留纯文本内容
- line:medium 仅在设置了 line 属性时生效
- empty 属性在内容完全为空或只包含空白字符时触发
