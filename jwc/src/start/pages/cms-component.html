cms-component标签
cms-component 开发模式中使用此组件进行代码块的引用，使用url属性引入在线代码块，子组件中的所有无data属性的组件，自动注入dataId。
属性
属性名称 说明 默认值
data 对应后台中list中的序列号，加载层级最高。 null
url 将在线的代码块引用到此组件中 null
innerText 引入代码块，只能在前端开发模式中使用 null
data加载顺序:cms-list.data   >   cms-component .data  >  url?data=dataId  >   cms-list .dev

注意事项

代码块的代码引入是单纯的内容引入，不是和vue一样将代码编译后的引入，所以对JavaScript比较不友好。

建议在一个页面多次引用同一个包含单个JavaScript组件时，使用ES6语法，参考以下代码：
new class component{
//你的代码逻辑
}
对于方法加载顺序对页面的影响，可以监听cms事件，这个事件将在所有的cms组件加载完毕后触发，用例：
document.addEventListener('cms' , ()=>{
//你的代码逻辑
})
