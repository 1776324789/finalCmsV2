cms-list标签
cms-list 加载列表数据
cms-list属性
属性名称 说明 默认值
data 对应后台中list中的序列号，加载层级最高。 null
dev 对应后台中list中的序列号，加载层级最低。 null
pagetitle 是否将此栏目的名称引用为页面title null
data加载顺序:cms-list.data   >   cms-component .data  >  url?data=dataId  >   cms-list .dev
cms-list子标签
子标签名称 功能 注意事项
list-name 加载列表的名称 无
list-note 加载列表的备注 无
list-cover 加载列表的封面 自身可以看作为img标签，所有写在该标签的style、class都会继承到创建的img子标签中，然后移除自身标签。
list-node-size 包含内容的数量 无
list-template 跳转到列表页面 该标签包裹的部分点击后跳转
list-list 加载列表的子列表 通过创建cms-list的方式加载子列表，可嵌套。（cms-list参与加载，但不参与到最终的渲染）
list-node 加载列表的子内容列表 通过创建cms-node的方式加载子内容，可嵌套。（cms-node参与加载，但不参与到最终的渲染）
list-pagination 子内容列表中的翻页组件 该标签只能在list-node中使用
所有写在页面中的组件标签都可以视作一个div，组件标签自身不带任何样式。

属性名称 说明 默认值
page 页面序号 1
limit 每页的加载数量 10
order 按照时间进行排序,desc:按照时间倒序，asc:按照时间正序 desc
top 是否使用置顶机制 false
newpage 是否在新的页面进行翻页 false
list-node中会自动创建需要加载的node节点，并且创建相应的cms-node，然后在cms-node加载前，将除list-pagination相关元素外的所有元素克隆至创建的每个cms-node中，
但是cms-node标签在加载后会将自身在list-node中移除，不作为页面元素。
